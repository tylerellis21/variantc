====== Functions ======
--------------------
 
Syntax:
def <identifier>(<arguments...>) <return type> { <stmt-body> }
def <identifier>(<arguments...>)  <return type> <identifer> { <stmt-body> }

Example:
def foo(i32 a, i32 b) i32 { return x + y; }
